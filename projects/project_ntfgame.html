NFT Collection Relesase

Release on Open Sea of Non-Fungible Tokens on the Polygon Blockchain.
Buyable NFTs that allow the redemption of Items within a mobile game.

The collection of NFTs will be released on the Polygon's Blockchain, chosen for its very affordable cost and speed.
These NFTs will be tied to representations of characters within the game in development by the XXXXX house. They will be characterized with various skins and accessories that will follow a set of items of a certain rarity, combinations of this sets i decided through an algorithm written in Python that will be, later, published as an open-source resource along with the seed, which created the various combinations, and the rarity % set in the algorithm of the individual traits.
In addition, each character will be unique and through its NFT, it can be redeemed in the store of the game, along with the traits that accompany it. The development of the game will be entrusted entirely to the house, so the main mechanics are not defined yet, but we can say that the game will be inspired by the now classic "Flappy Bird", all with under the light of an obstacle course with some item-in-game like "Mario Kart" style to interact with opponents.
The game will be on mobile and will be developed in Unity, the release is expected about a year after the release of NFT.
